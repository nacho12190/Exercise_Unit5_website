# Exercise _Unit5_website

## Repository Description

Here are the files for a simple website I created for the Unit 5 exercise on version control with Git and GitHub.

## Technologies Used

* **HTML**: For the content structure (`index.html`).

* **CSS**: For the style and appearance of the page (`styles.css`).

## How to Open the Website

To view the page on your computer:

1. Clone this repository to your machine.

2. Open the **`index.html`** file in your preferred browser.
